# readablecode.github.io

The static, GitHub-hosted mirror of my portfolio — an interactive, terminal-first
page (type `help`) with a project browser, a `herdstone`-style herd-monitor
animation, and a skills overview. Pure HTML/CSS/JS, no build step.

The live, dynamically-served twin (React + a real FastAPI backend, self-hosted on
my own homelab) lives at **[site.tinkernet.me](https://site.tinkernet.me)**.

- 🌐 Live site: <https://site.tinkernet.me>
- 🐙 GitHub: <https://github.com/ReadableCode>
